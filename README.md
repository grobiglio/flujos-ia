# 🧠 Mis Flujos de Trabajo con IA

Este repositorio centraliza y documenta mis herramientas y flujos de trabajo basados en Inteligencia Artificial. El objetivo principal es mantener un control de versiones simple pero estricto sobre las instrucciones, contexto y reglas que utilizo al interactuar con distintos modelos de lenguaje.

## 🎯 Propósito
La IA es muy sensible a pequeños cambios en el texto. Utilizo este repositorio para iterar y perfeccionar:
* **Prompts:** Optimizados para tareas específicas (programación, educación, análisis de ingeniería, etc.).
* **OpenAI GPTs:** Instrucciones base (System Prompts) y registro de archivos de conocimiento.
* **Gemini Gems:** Configuraciones de comportamiento y tono.

Al usar Git y archivos `.md`, puedo rastrear exactamente qué modificación mejoró (o empeoró) la respuesta del modelo a lo largo del tiempo.
